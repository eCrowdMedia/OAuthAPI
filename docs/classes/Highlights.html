Parameters:

  • [options] Object optional
    • readingId String

      The numerical id of the desire resource.

    • [highlight[contet] Number optional

      The content of the highlight.

    • [highlight[locators] String optional

      Locators are used to determine the exact location of the highlight in a larger piece of text. This is a custom JSON structure that contains a few different data points.

    • [highlight[position] String optional

      The position in the book where this highlight was made as a value 0.0 - 1.0.

    • [highlight[highlighted_at] String optional

      The time the highlight was created. This is used to create highlights after they were posted. Recommended date format is ISO 8601.

    • [highlight[post_to[] String optional

      This parameter is used for sharing to other networks.

    • [comment[content] String optional

      The comment text.
