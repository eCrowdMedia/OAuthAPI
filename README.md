Readmoo OAuth API
=================

This is an oauth api library for readmoo api.


Install
=======

Install from bower,

```bash
bower install readmoo-oauth-api --save
```


and add srcipt into html.

```html
<script src="bower_components/readmoo-oauth-api/dist/readmoo-oauth-api.min.js"></script>
```
